# Task Manager Client

## Requirements

`Nodejs`

`NPM`

## Installation & Running the App

Clone the project

```
git clone https://github.com/EssaadaniYounes/task-manager-client
```

Install the dependencies

```
npm install
```

Run the app in dev

```
npm run dev
```
